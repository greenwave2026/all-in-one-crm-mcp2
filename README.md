# All-in-One CRM MCP Server

Read-only MCP server for Zigaflow, Spruce, and ServiceM8.

## Tools

- `list_connected_crms`
- `search_crm_records`
- `get_pipeline_summary`
- `get_recent_activity`
- `get_followups_due`
- `get_daily_digest`
